# ColorBridge Demo 历史批次参数数据

## 使用说明

这份数据用于 ColorBridge Demo 的历史订单匹配、参数微调、风险预警和方案卡展示。

数据不是可直接投产的真实工厂配方，而是根据公开资料中的常见工艺范围，生成的演示用历史批次。真实生产必须由工艺员结合染料品牌、面料前处理、设备、浴比、客户标准和实验室小样确认。

## 公开资料依据

本数据采用以下公开资料中的行业常见范围作为生成依据：

- 棉/纤维素活性染料：常见固色 pH 约 10.5-11.2；温染/热染活性染料常在 60-80°C；盐、碱用量随深度变化。
- 涤纶分散染料：高温高压染色常在 130-135°C；弱酸性 pH 约 4.5-5.5；常配分散剂、匀染剂。
- 锦纶酸性染料：常在酸性条件 pH 4.5-5.5；90-100°C 保温 30-60 分钟；pH 过低或升温过快容易不匀。
- 腈纶阳离子染料：常在弱酸性 pH 4-5.5；95-100°C 染色；升温速率需要控制，避免色花。

## 字段说明

| 字段 | 说明 |
|---|---|
| `batch_id` | 历史批次号 |
| `fabric` | 面料 |
| `composition` | 纤维组成 |
| `weight_gsm` | 克重 |
| `color` | 颜色名称 |
| `dye_type` | 染料体系 |
| `target_lab` | 目标 Lab |
| `actual_lab` | 实际 Lab |
| `delta_e` | 历史色差 |
| `rft` | 是否一次成功 |
| `reworked` | 是否回修 |
| `formula` | 染料配方 |
| `process` | 工艺参数 |
| `result_note` | 结果说明 |

## Demo 风险规则建议

| 参数偏离 | 建议提示 |
|---|---|
| 温度高于成功区间 | 可能导致上染增强、颜色偏深；锦纶和腈纶还可能增加不匀风险。 |
| pH 高于成功区间 | 活性染料可能固色过快、偏深或不匀；酸性/阳离子体系可能降低正常上染控制。 |
| pH 低于成功区间 | 酸性/阳离子体系可能上染过快，增加色花；部分纤维有损伤风险。 |
| 盐/元明粉偏高 | 棉活性染料促染增强，可能偏深。 |
| 纯碱偏高 | 棉活性染料固色增强但水解风险上升，可能偏暗或不匀。 |
| 浴比偏大 | 染液浓度下降，可能偏浅；但浅色匀染性可能更好。 |
| 升温过快 | 分散、酸性、阳离子体系容易色花、条花或局部偏深。 |
| 保温不足 | 可能上染不充分，颜色偏浅或牢度不足。 |
